### Getting Started

When you first open the application, you'll see an empty pyramid with a navigation bar at the top. The main view shows your "Not-To-Do" items arranged in a pyramid structure.

### Main Interface

#### Navigation Bar
- **金字塔 (Pyramid)**: Main view showing your not-to-do items
- **黑洞审计 (Black Hole Audit)**: Guided wizard for analyzing failures
- **数据管理 (Data Management)**: Import/export and backup options

#### Pyramid View
- Items are arranged in rows (1, 2, 3, 4... items per row)
- Each item has a color based on its category
- Newest items appear at the top of the pyramid

### Adding Items

1. Click the "添加'不为'准则" (Add Not-To-Do Rule) button at the bottom
2. Select a category:
   - **深度工作 (Deep Work)**: Professional boundaries
   - **健康 (Health)**: Health-related boundaries
   - **社交 (Social)**: Social boundaries
   - **财务 (Finance)**: Financial boundaries
   - **其他 (Other)**: Miscellaneous boundaries
3. Enter your rule in the text field
4. Add a reflection (optional but recommended):
   - Describe a past experience related to this rule
   - This helps reinforce why this boundary is important
5. Click "确认不为" (Confirm) to add it

### Black Hole Audit

The audit wizard helps you turn failures into wisdom:

#### Step 1: Describe the Failure
- What went wrong?
- What activity or behavior caused the problem?
- Be specific and honest

#### Step 2: Identify Root Causes
- Why did this happen?
- What were the underlying factors?
- Consider emotional, environmental, and habitual causes

#### Step 3: Create a Rule
- Based on your analysis, create a specific "not-to-do" rule
- Make it actionable and clear
- Select the most appropriate category

The wizard automatically adds the new rule to your pyramid with the audit details as a reflection.

### Managing Your Pyramid

#### Viewing Items
- Hover over any item to see its full text
- Items with reflections show a clock icon
- Hover over items with reflections to see the details

#### Deleting Items
- Hover over an item to reveal the trash icon
- Click the trash icon to delete the item
- Confirm the deletion if prompted

#### Categories and Colors
- Deep Work: Violet border
- Health: Emerald border
- Social: Blue border
- Finance: Amber border
- Other: Stone border

### Data Management

#### Export Options
1. Click "数据管理" (Data Management) in the navigation
2. Choose your export format:
   - **JSON**: Complete backup for restoration
   - **Markdown**: Formatted document for sharing

#### Import Options
1. In Data Management, click "Import JSON"
2. Select your backup file
3. Confirm to replace current data

#### Local Storage
- All data is automatically saved to your browser's local storage
- Clearing browser data will remove your not-to-do list
- Regular backups are recommended

### Tips and Best Practices

1. **Be Specific**: Instead of "Don't waste time", use "Don't check social media before 10 AM"
2. **Add Reflections**: They provide motivation and context
3. **Regular Audits**: Use the audit wizard weekly to process recent failures
4. **Keep it Balanced**: Ensure you have boundaries across all life areas
5. **Review Regularly**: Periodically review and update your list
6. **Backup Often**: Export your data regularly to prevent loss

### 开始使用

首次打开应用时，您会看到一个空的金字塔，顶部有导航栏。主视图显示您的"不为"项目，以金字塔结构排列。

### 主界面

#### 导航栏
- **金字塔 (Pyramid)**: 显示您的"不为"项目的主视图
- **黑洞审计 (Black Hole Audit)**: 分析失败的引导向导
- **数据管理 (Data Management)**: 导入/导出和备份选项

#### 金字塔视图
- 项目按行排列（每行1、2、3、4...个项目）
- 每个项目根据其分类有不同的颜色
- 最新项目出现在金字塔顶部

### 添加项目

1. 点击底部的"添加'不为'准则"按钮
2. 选择一个分类：
   - **深度工作**: 专业边界
   - **健康**: 健康相关边界
   - **社交**: 社交边界
   - **财务**: 财务边界
   - **其他**: 其他边界
3. 在文本字段中输入您的规则
4. 添加反思（可选但推荐）：
   - 描述与此规则相关的过往经历
   - 这有助于强化为什么这个边界很重要
5. 点击"确认不为"添加

### 黑洞审计

审计向导帮助您将失败转化为智慧：

#### 第1步：描述失败
- 出了什么问题？
- 什么活动或行为导致了问题？
- 具体且诚实地描述

#### 第2步：识别根本原因
- 为什么会发生这种情况？
- 潜在因素是什么？
- 考虑情感、环境和习惯性原因

#### 第3步：创建规则
- 基于您的分析，创建具体的"不为"规则
- 使其可操作且清晰
- 选择最合适的分类

向导会自动将新规则添加到您的金字塔中，并将审计详情作为反思。

### 管理金字塔

#### 查看项目
- 悬停在任何项目上查看其完整文本
- 有反思的项目显示时钟图标
- 悬停在有反思的项目上查看详情

#### 删除项目
- 悬停在项目上显示垃圾桶图标
- 点击垃圾桶图标删除项目
- 如果提示，确认删除

#### 分类和颜色
- 深度工作: 紫色边框
- 健康: 翠绿色边框
- 社交: 蓝色边框
- 财务: 琥珀色边框
- 其他: 石灰色边框

### 数据管理

#### 导出选项
1. 在导航中点击"数据管理"
2. 选择导出格式：
   - **JSON**: 用于恢复的完整备份
   - **Markdown**: 用于分享的格式化文档

#### 导入选项
1. 在数据管理中，点击"导入JSON"
2. 选择您的备份文件
3. 确认替换当前数据

#### 本地存储
- 所有数据自动保存到浏览器的本地存储
- 清除浏览器数据将删除您的"不为"清单
- 建议定期备份

### 技巧与最佳实践

1. **具体明确**: 不要说"不要浪费时间"，而是说"上午10点前不要查看社交媒体"
2. **添加反思**: 它们提供动力和背景
3. **定期审计**: 每周使用审计向导处理最近的失败
4. **保持平衡**: 确保在所有生活领域都有边界
5. **定期审查**: 定期审查和更新您的清单
6. **经常备份**: 定期导出数据以防丢失

## Troubleshooting

### Common Issues

1. **Data not saving**
   - Check if browser allows local storage
   - Try refreshing the page
   - Export and re-import your data

2. **Items not displaying correctly**
   - Refresh the page
   - Check your browser version
   - Try a different browser

3. **Import/Export not working**
   - Ensure you're using a modern browser
   - Check file permissions
   - Verify file format is correct JSON

### Getting Help

If you encounter issues:
1. Check this guide first
2. Export your data as a backup
3. Clear browser cache and try again
4. Open an issue on GitHub with details

## FAQ

**Q: Can I use this app offline?**
A: Yes, once loaded, the app works offline. Data syncs when online.

**Q: Is my data private?**
A: Yes, all data is stored locally in your browser. Nothing is sent to servers.

**Q: Can I share my not-to-do list?**
A: Use the Markdown export feature to create a shareable document.

**Q: How many items can I add?**
A: There's no hard limit, but performance may degrade with very large lists.

**Q: Can I customize categories?**
A: Currently not, but this is planned for a future update.

## 常见问题

**问：我可以离线使用这个应用吗？**
答：可以，一旦加载，应用就可以离线工作。在线时数据会同步。

**问：我的数据是私密的吗？**
答：是的，所有数据都本地存储在您的浏览器中。没有任何内容发送到服务器。

**问：我可以分享我的"不为"清单吗？**
答：使用Markdown导出功能创建可分享的文档。

**问：我可以添加多少个项目？**
答：没有硬性限制，但非常大的列表可能会影响性能。

**问：我可以自定义分类吗？**
答：目前不可以，但这是未来更新计划中的功能。
