# 不为清单金字塔 (Not-To-Do List Pyramid)

## English

### 🌟 Vision

Not-To-Do List Pyramid is a visual tool based on the "Not-To-Do List" principle. By clearly defining behavioral boundaries, it helps reduce life detours and accumulate long-term positive value.

**"Not-To-Do" is for better "To-Do".** By establishing clear boundaries, we minimize distractions and maintain the foundation of long-term growth.

### 🚀 Key Features

#### 📊 Pyramid Visualization
- Intuitively display your behavioral boundaries in a pyramid structure
- Visual hierarchy that emphasizes the most important "not-to-do" items
- Animated interactions for better user experience
- Items are arranged in rows: 1 item in the first row, 2 in the second, and so on

#### 🏷️ Category Management
Organize your not-to-do items into five categories:
- **Deep Work** (深度工作): Professional tasks and career development
- **Health** (健康): Physical and mental wellness
- **Social** (社交): Relationships and networking
- **Finance** (财务): Financial decisions and spending habits
- **Other** (其他): Miscellaneous items

Each category has its own color scheme and icon for easy identification.

#### 🔍 Energy Black Hole Audit
A guided wizard to help you:
- Review past failures and mistakes
- Identify root causes of energy drains
- Extract valuable lessons to create new not-to-do rules
- Transform painful experiences into wisdom

The audit process consists of:
1. Describing a recent failure or energy drain
2. Analyzing the root causes
3. Creating a specific "not-to-do" rule
4. Automatically adding it to your pyramid

#### 💾 Data Management
- **JSON Backup/Restore**: Full data backup and restore functionality
- **Markdown Export**: Export your not-to-do list as a formatted markdown document
- **Local Storage**: Data persists in browser's local storage
- **Import/Export**: Easy data migration between devices

#### 📱 Responsive Design
- Optimized for both desktop and mobile devices
- Smooth animations and transitions
- Dark theme with glassmorphism effects
- Touch-friendly interface for mobile users

### 🛠 Tech Stack

- **Frontend Framework**: React 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Animations**: Motion (Framer Motion)
- **TypeScript**: Full type safety

### 📦 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sdlyingyong/growth-roadmap.git
   cd growth-roadmap
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Access the application**:
   Open your browser and navigate to `http://localhost:3000`

### 🎯 How to Use

#### Adding Not-To-Do Items
1. Click the "添加'不为'准则" (Add Not-To-Do Rule) button at the bottom
2. Select a category for your item from the available options
3. Enter the not-to-do rule (e.g., "Don't scroll social media for more than 30 minutes")
4. Optionally, add a reflection about past experiences (this appears as a tooltip)
5. Click "确认不为" (Confirm) to add it to your pyramid

#### Using the Audit Wizard
1. Navigate to the "黑洞审计" (Black Hole Audit) tab
2. Follow the step-by-step wizard to analyze past failures:
   - **Step 1**: Describe a recent failure or energy drain
   - **Step 2**: Identify the root causes
   - **Step 3**: Create a specific rule based on your analysis
3. The wizard will automatically add the new item to your pyramid

#### Managing Your Pyramid
- **View Items**: Items are displayed in a pyramid structure with the newest at the top
- **Delete Items**: Hover over any item and click the trash icon to remove it
- **View Reflections**: Hover over items with a clock icon to see past reflections
- **Categories**: Each item is color-coded by category

#### Data Management
1. Click "数据管理" (Data Management) in the navigation bar
2. **Export Options**:
   - **JSON**: Full backup of all data
   - **Markdown**: Formatted document for sharing or printing
3. **Import Options**:
   - **JSON**: Restore from a previous backup

1. 在导航栏中点击"数据管理"
2. **导出选项**：
   - **JSON**: 所有数据的完整备份
   - **Markdown**: 用于分享或打印的格式化文档
3. **导入选项**：
   - **JSON**: 从之前的备份恢复

## Philosophy Behind "Not-To-Do List"

The concept of "Not-To-Do List" is based on the principle that knowing what **not** to do is as important as knowing what to do. By explicitly defining our boundaries and eliminating energy drains, we create space for meaningful work and personal growth.

This tool helps you:
- Identify patterns of self-sabotage
- Learn from past mistakes
- Build sustainable habits
- Focus on what truly matters

Remember: Every "no" to a distraction is a "yes" to your goals.

---

## "不为清单"背后的理念

"不为清单"的概念基于一个原则：知道**不**该做什么和知道该做什么同样重要。通过明确定义我们的边界并消除能量消耗，我们为有意义的工作和个人成长创造空间。

这个工具帮助你：
- 识别自我破坏的模式
- 从过去的错误中学习
- 建立可持续的习惯
- 专注于真正重要的事情

记住：对干扰说"不"就是对目标说"是"。
